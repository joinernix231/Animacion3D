Sistema Solar 3D — Three.js
================================

Descripción
-----------
Visualización 3D simple del sistema solar hecha con HTML, CSS y Three.js. Permite explorar la escena, ver información básica de los planetas y ajustar la animación.

Cómo ejecutar
-------------
Opción rápida: abre `index.html` en el navegador.

Opción recomendada (servidor local):
```bash
python -m http.server 8000
# o
npx http-server
```
Luego visita `http://localhost:8000`.

Uso
---
- Rotar cámara: click izquierdo + arrastrar
- Zoom: rueda del mouse
- Pan: click derecho + arrastrar
- Click en planeta: muestra info
- Controles (panel derecho): velocidad, mostrar órbitas, mostrar nombres
